# QuantLab

QuantLab is a quantitative finance project focused on portfolio analytics, optimization, and strategy backtesting.

The goal of this project is to gradually evolve into a full-stack platform for quantitative finance research and portfolio analysis.

---

## Current Features

### Portfolio Optimization

- Historical stock data using yfinance
- Log return computation
- Expected return estimation
- Covariance matrix computation
- Minimum Variance Portfolio (MVP)
- Efficient Frontier visualization
- Maximum Sharpe Portfolio
- PCA-based analysis of asset returns

### Strategy Backtesting

- SMA crossover strategy
- EMA crossover strategy
- Annualized return
- Annualized volatility
- Sharpe Ratio
- Maximum Drawdown
- Transaction cost analysis
